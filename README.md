# Preemptive Priority CPU Scheduling in C

This project implements the **Preemptive Priority CPU Scheduling Algorithm** using the C programming language.

In this scheduling method, the process with the **highest priority value gets executed first**. If a new process arrives with a **higher priority**, it will **preempt the currently running process**.

## Features
- Implements **Preemptive Priority Scheduling**
- Accepts process details as input
- Calculates:
  - Waiting Time
  - Turnaround Time
  - Average Waiting Time
  - Average Turnaround Time
